# youtube-rip-py

## Introduction
`youtube-rip-py` is a small ported Python program that uses `yt-dlp` to rip the audio from YouTube videos and save them locally, allowing you to save songs and podcasts locally with ease. It's made to be compatible with all machines while still providing the same functionality as [youtube-rip](https://github.com/mr-sunset/youtube-rip).

## Installation
`youtube-rip-py` depends on yt-dlp installed via Homebrew.

Download Homebrew (Mac):
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Download Python:
```bash
brew install python
```

Download `yt-dlp`: 
```bash
brew install yt-dlp
```

Download the script (download raw file):

[Raw code (GitHub)](https://github.com/mr-sunset/youtube-rip-py/blob/main/youtube-rip.py)
